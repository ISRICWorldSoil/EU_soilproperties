# EU_soilproperties
Mapping EU soil properties using point data LUCAS and GEMAS

Targeted soil properties:
  - soil organic carbon,
  - soil pH in H2O and CaCl2,
  - soil texture fractions sand silt and clay,
  - heavy metals As, Cd, Cu, Pb, Zn,
